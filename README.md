# System-Heat-Extensions
System Heat Extensions is a mod that extends the System Heat environment to include thermoelectrics, thermal nozzles, and more! I do not plan on adding parts for now, and this is very much in beta.

Built for Kerbal Space Program 1.12.5 using the SystemHeat framework. Thanks to Nertea and the KSP Mod Stewards for the SystemHeat projects that provide the thermal infrastructure this mod integrates with.
